# Colmar Academy — Capstone Project

Landing page for a fictional school, built as the capstone for Codecademy's **"Build a Website with HTML, CSS and GitHub Pages"** path.

**Live site:** https://costa911.github.io/colmar-academy/

## Why I built this

At work I've been using HTML to build lightweight dashboards for stakeholders who need fast, visual answers without BI tool access or permissions. This course was about sharpening that skill — specifically, learning HTML/CSS well enough that I'm directing the code, not just prompting my way to something that works.

AI writing HTML/CSS at speed is the real unlock. But understanding what's underneath it is what lets a small change — a font color, a layout tweak — stay a small change instead of turning into a full regeneration.

## Built from

- Wireframes and image/video assets provided by Codecademy
- Semantic HTML5 + CSS (Grid/Flexbox, media queries for responsive breakpoints)
- No frameworks — vanilla HTML/CSS by design, to keep the fundamentals visible
